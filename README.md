# Sistema de Gimnasio 🏋️‍♀️💪

Proyecto final de Programación 1 desarrollado en Python utilizando Programación Orientada a Objetos (POO).

## Descripción

Este programa simula un sistema básico de administración de gimnasio.  
El sistema permite registrar miembros, entrenadores, visualizar planes de membresía y realizar inscripciones con métodos de pago.

## Funciones

- Agregar miembros
- Agregar entrenadores
- Mostrar miembros registrados
- Mostrar entrenadores registrados
- Mostrar planes disponibles
- Crear inscripciones
- Procesar pagos en efectivo y tarjeta

## Conceptos aplicados

En este proyecto se aplicaron los siguientes conceptos vistos en clase:

- Clases
- Objetos
- Métodos
- Método constructor (`__init__`)
- Instancia de objetos
- Listas
- Menús interactivos
- Relaciones entre clases
